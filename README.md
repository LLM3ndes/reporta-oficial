# Reporta

Aplicativo desenvolvido para gestão de problemas urbanos, permitindo que cidadãos registrem ocorrências em tempo real.

## Tecnologias Utilizadas
- React Native (Expo)
- Firebase Firestore

## Funcionalidades
- Listagem de ocorrências em tempo real;
- Formulário para envio de novos problemas;
- Interface intuitiva e responsiva.

## Como Executar
1. Clone o repositório;
2. Instale as dependências: `npm install`
3. Inicie o projeto: `npx expo start`
