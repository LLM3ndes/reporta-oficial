Reporta: Transformando nossa cidade juntos
O Reporta é uma ferramenta de tecnologia desenvolvida para aproximar o cidadão da gestão pública. O objetivo é simples: permitir que você ajude a cuidar do lugar onde vive, registrando problemas urbanos (como um buraco na rua ou uma lâmpada queimada) de forma rápida e transparente.

Por que o Reporta é importante?
Imagine que a sua rua tem um problema que precisa de atenção. Com o Reporta, você não apenas identifica essa necessidade, mas garante que ela seja vista e acompanhada, transformando a colaboração de cada um em um benefício real para toda a vizinhança. É tecnologia sendo usada para melhorar a nossa qualidade de vida.

O que o app faz?
Seu Registro em Tempo Real: Assim que você envia um problema, ele aparece em nossa lista, garantindo transparência e organização.

Categorização Inteligente: Organizamos as denúncias por tipo (Iluminação, Buraco, Lixo, etc.) para que a solução chegue mais rápido a quem precisa.

Acompanhamento de Status: Você sabe exatamente quando um problema está "Pendente" ou quando já foi "Resolvido".

Como começar a usar?
Desenvolvemos o Reporta para ser fácil de usar por qualquer pessoa. Siga estes passos para ter o projeto rodando:

Obter o código: Clone este repositório para o seu computador.

Preparar o terreno: Dentro da pasta do projeto, abra o terminal e digite npm install. Isso vai baixar todos os "blocos de montar" que o app precisa para funcionar.

Ligar o motor: Digite npx expo start --web no seu terminal ou use o app expo go. Agora é só abrir o app e começar a transformar sua cidade!
